# 谜题说明

## 谜面

没有谜面，只能提交。

玩家必须在 UNIX 时间的「质数秒」点击提交按扭（因为信息发送到服务器可能有延迟，所以有 5 秒的可接受误差），连续且不重复 4 次（如果遗漏了一个质数秒，或者在同一秒内点击了多次，都算作错误）。

- 如果出现不符合要求的提交，提示报错 `NulliPrime`；
- 如果符合要求的提交达到一次，提示报错 `SimPrime`；
- 如果符合要求的提交达到两次，提示报错 `BiPrime`；
- 如果符合要求的提交达到三次，提示报错 `TerPrime`；
- 如果符合要求的提交达到三次，通关并提示 `QuaterPrime`。

## 题解

nulli-, sim-, bi-, ter-, quater-分别是拉丁语次数前缀词 0, 1, 2, 3, 4。因此 QuaterPrime 意为“4 次质数”。
